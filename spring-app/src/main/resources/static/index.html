Invoice Matcher (Kotlin)

Déposez vos factures (PDF) et un relevé bancaire, puis lancez l'analyse. Les fichiers sont envoyés à l'API /api/matching/run.

Formats attendus : factures PDF ; relevé en PDF, CSV (séparateur ,, ;, tabulation ou |) ou Excel (XLS/XLSX). Le relevé doit contenir des colonnes date, description et montant. Avec l'option « Utiliser Ollama », l'IA tente d'identifier automatiquement les colonnes même si les entêtes sont exotiques (multi-langues, arabe, etc.). Un message d'erreur détaillé s'affichera si rien n'est détecté.
